AI Based Lecture Summarizer & Quiz Generator

Lectures are often recorded or delivered online, enabling students to access them anytime. However, despite this accessibility, students face difficulties in retaining large volumes of lecture material, especially when lectures are long, dense, or delivered at a fast pace. Thus, there is a pressing need for an AI‑based system capable of:
  • Automatically transcribing lecture audio/video,
  • Generating high‑quality summaries, and
  • Creating quiz questions that support revision and learning.

Problem Statement

Students face difficulty in retaining long and dense lecture content due to the lack of automated tools that convert classroom lectures into concise summaries and quizzes. The project aims to develop an AI‑based system that can automatically transcribe, summarize, and generate quizzes from lecture audio/video using advanced NLP and ASR models.

LITERATURE REVIEW / MARKET RESEARCH

Research shows that AI lecture summarizers and quiz generators leverage NLP and LLMs to extract key concepts and create automated assessments, improving learning efficiency and engagement. Challenges include maintaining accuracy, contextual understanding, and ensuring reliable outputs.

RESEARCH GAP / INNOVATION

The system integrates lecture transcription, AI summarization, and automated quiz generation with an intuitive interface to enhance learning effectiveness and engagement.

SYSTEM METHODOLOGY

DATASET / INPUT

Lecture audio, video, PDF, and text files. Supports MP3, WAV, MP4, WebM, and document formats.

MODEL / ARCHITECTURE

Speech-to-text (Whisper/ASR), LLM summarization, and quiz generation pipeline with REST API.

LIVE EXECUTION

Interactive web application with upload, transcript, summary tabs, and quiz generation.

Run npm run dev in the lecture-mind folder first.
