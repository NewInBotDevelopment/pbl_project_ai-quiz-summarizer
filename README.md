# PBL 2026 — AI Based Lecture Summarizer & Quiz Generator

Project showcase webpage for the PBL (Project-Based Learning) submission.

## How to View

1. **Open the PBL page:**  
   Double-click `index.html` or open it in your browser.  
   Or run a local server:  
   `npx serve .` (if you have Node.js)

2. **View the Live Demo:**  
   The "VIEW CODE / DEMO" button links to the Lecture Mind app.  
   For it to work, run the app first:
   ```bash
   cd ../lecture-mind
   npm install
   npm run dev
   ```
   Then the button will open http://localhost:3000

## Navigation

- **PROBLEM** — Problem Statement, Literature Review, Research Gap
- **METHODOLOGY** — System Methodology (Dataset, Model, Live Demo)
- **RESULTS** — Results & Analysis with bar chart
- **TEAM** — Academic Credits (Guide + Team Members)
