Lecture Mind demo

Start the app, then open the demo.

The demo runs as a separate app on your computer. If you see "This site can't be reached", the app isn't running yet. Follow these steps:

  1. Open a terminal or Command Prompt.
  2. Go to the project folder and into lecture-mind:
    cd lecture-mind
  3. Install dependencies (first time only):
    npm install
  4. Start the app:
    npm run dev
  5. When you see "Ready" in the terminal, click the button below.
Open Lecture Mind →

You're viewing this page on the live site. The interactive Lecture Mind demo runs locally or on a separate host.

To run the demo yourself:

  1. Clone or download this repository.
  2. Open a terminal in the project folder, then run:
    cd lecture-mind
    npm install
    npm run dev
  3. Open http://localhost:3000 in your browser.
